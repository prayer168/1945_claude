# STRIKER 1945 — 仿《1945》射擊遊戲

純 HTML + Canvas + 原生 JavaScript 打造的垂直捲軸射擊遊戲，零外部資源，單檔即可遊玩。

## 🎮 線上遊玩

## 操作方式
| 動作 | 鍵盤 | 觸控 |
|---|---|---|
| 移動 | 方向鍵 / `W A S D` | 拖曳畫面 |
| 射擊 | 自動發射 | 自動發射 |
| 炸彈 | `空白` 或 `B` | 雙擊畫面 |

## 遊戲特色
- **6 大關卡**，每關專屬背景色調、雜兵波次由易到難
- **6 隻 Boss**，招式各異（扇形、環形、旋轉螺旋、雙向反轉環、最終高密度彈幕）
- 寫實俯視戰機外型（旋轉螺旋槳、引擎尾焰、雙引擎轟炸機等）
- 道具系統：`P` 火力升級（最高 4 級散射）、`B` 加炸彈、`♥` 補血
- 分數 / 生命 / 螢幕震動 / 粒子爆炸 / 純程式 WebAudio 音效
- 本機排行榜（前 5 名，存 localStorage）
